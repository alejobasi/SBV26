# Journal of Resolved Problems

> Persistent memory for non-trivial technical problems already solved in
> this project.
>
> Read this file before entering a long debug loop. Update it when a
> difficult issue is finally resolved.

---

## Usage Rules

- Add an entry only when the problem was non-trivial or required several
  iterations.
- Prefer updating an existing entry if the root cause is the same.
- Keep the exact error text and reusable keywords.
- Record only validated solutions.
